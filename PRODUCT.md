# Product

## Users

开发者与新贡献者第一次接触陌生仓库时使用：需要尽快建立可信的心智模型，并知道先读哪些源码。

## Purpose

把一个仓库地址变成**一份**可离线打开的精致学习页。用源码证据讲清目标、结构、流程、概念与学习顺序——不是把 README 重新排版，也不是审计报告生成器。

## Personality

冷静、精确，像一份可印刷的深度简报。气质对齐 **浅色编辑部研报**：纸色底、青绿强调、衬线标题、发丝线分区、Mermaid 嵌在浅底 panel。

## Principles

1. 首屏立刻回答「这是什么」——项目名是 masthead 级信号。
2. 强层级靠排版与发丝线，不为了「干净」而发灰发平，也不靠暗色发光堆氛围。
3. 图与文字相邻、互相印证；图板融入纸色主题。
4. 自顶向下；少而准。
5. 单文件交付；模板负责气质，Agent 负责理解。

## Anti-goals

- JSON schema / 多层校验流水线作为主路径
- 无样式「语义站」冒充产品
- 无证据的示意图与空洞指标
- AI 营销风（紫渐变、毛玻璃卡片墙、三列概念卡）
- 发灰发平的「说明书默认皮」
- 夜间霓虹 / 铜光发光图鉴冒充研报
