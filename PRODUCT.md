# Product

## Users

开发者与新贡献者第一次接触陌生仓库时使用：需要尽快建立可信的心智模型，并知道先读哪些源码。

## Purpose

把一个仓库地址变成**一份**可离线打开的精致学习页。用源码证据讲清目标、结构、流程、概念与学习顺序——不是把 README 重新排版，也不是审计报告生成器。

## Personality

冷静、精确，但视觉上要有存在感。气质对齐 **夜间工程图鉴**：深底、铜光强调、大标题压场、Mermaid 图板发光。

## Principles

1. 首屏立刻回答「这是什么」——项目名是英雄级信号。
2. 强层级与氛围，不为了「干净」而发灰发平。
3. 图与文字相邻、互相印证；图板融入暗色主题。
4. 自顶向下；少而准。
5. 单文件交付；模板负责气质，Agent 负责理解。

## Anti-goals

- JSON schema / 多层校验流水线作为主路径
- 无样式「语义站」冒充产品
- 无证据的示意图与空洞指标
- AI 营销风（紫渐变、毛玻璃卡片墙、三列概念卡）
- 发灰发平的「说明书默认皮」
